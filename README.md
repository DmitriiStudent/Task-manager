# Task-manager
Простое приложение для управления задач. Проект, созданный в целях обучения.

Это небольшое приложение для повседневных дел, которое я написал, чтобы разобраться с React. Оно помогает планировать задачи и привязывать их к конкретным датам.

Всё работает прямо в браузере. Вы выбираете день, добавляете задачи с названием и описанием, отмечаете выполненные или удаляете их. После перезагрузки страницы список не пропадает, потому что данные хранятся в локальном хранилище браузера.

Что использовалось в работе: React 18 с хуками useState и useEffect, обычный CSS для стилей и localStorage для хранения данных.

Чтобы запустить проект локально, нужно склонировать репозиторий, установить зависимости командой npm install, а затем запустить приложение через npm run dev. Оно откроется в браузере на локальном адресе.

Из того, что можно ещё добавить в будущем: редактирование уже созданных задач, фильтрацию по статусу и экспорт списка в файл.
